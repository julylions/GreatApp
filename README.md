# GreatApp

这个代码库将不断更新，用于实践优秀第三方框架的思想和原理

##简介

目前参考YYImage和Kingfisher用Swift写了一个图片加载功能，可以加载jpg、png、apng、gif、webp等图片格式

还有搜集整理的很多Extension功能
